## Описание репозитория
Данный репозиторий предназначен для хранения и выполнения лабораторных работ


## Структура репозитория
В репозитории содержатся следующие разделы:

- `lab1` — первая лабораторная работа
- `lab3` — третья лабораторная работа  
- `lab4` — четвертая лабораторная работа  

Каждая папка содержит исходный код и необходимые файлы для выполнения соответствующей лабораторной работы.
